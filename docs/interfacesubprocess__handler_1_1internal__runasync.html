internal_runasync
Handler

Definition

Interface for asynchronously running a subprocess. This generic interface provides a single procedure for executing a command asynchronously, allowing it to run in the background without waiting for completion.

integer function internal_runasync_default ( character(*), intent(in) cmd,
type(handle_pointer), intent(inout) fp,
integer(c_int), intent(out) excode )

Executes a command asynchronously without waiting for completion. This function creates a subprocess with asynchronous capabilities and returns immediately, allowing the command to run in the background.

Parameters
[in]cmdThe command line string to execute (e.g., "notepad.exe").
[in,out]fpThe handle pointer to manage the subprocess.
[out]excodeThe initial exit code (set to -1 if creation fails).
Returns
pid The process ID of the created subprocess, or a negative value on error.
