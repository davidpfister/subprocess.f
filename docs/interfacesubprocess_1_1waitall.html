waitall
Subprocess

Definition

Waits for all processes in an array to complete.

Methods

waitall(type(process) p(:))

Parameters
[in,out]processesArray of process objects to wait for.

Examples

The following example illustrates a call to the waitall interface.

type(process) :: p(5)
integer :: i
do i = 1, 5
p(i) = p = process('long_running_process')
call p(i)%run()
end do
call waitall(p)
