read_stderr Interface Reference
Overview » Subprocess

Definition

Reads the standard error of the associated process.

Methods

read_stderr(type(process) p(:), character(:), allocatable output)

Parameters
[in,out]thisThe process object.
[out]outputThe captured stderr as a character string.

Examples

The following example illustrates a call to the read_stderr interface. It uses the return code from the exit_code function to determine whether or not an error occurred during execution.

type(process) :: p
character(:), allocatable :: errmsg
type(process) :: p
p = process('ls')
call run(p, './ *.f90')
call read_stderr(p, errmsg)
if (p%exit_code() /= 0) then
call read_stderr(p, errmsg)
print *, errmsg
end if
