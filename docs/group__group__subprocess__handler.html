Overview

Introduction

Provides the binding and wrapper functions to the processlib C library.

Remarks

The binding is obtained using the iso_c_binding intrinsic module. If the compiler does not provide such intrinsic module then the compilation will fail.

Data Types

type  handle_pointer
 A derived type wrapping a subprocess handle to manage its lifecycle. This type encapsulates a subprocess_s handle, making it allocatable to avoid gfortran-specific segmentation faults during finalization (see Bug 82996). It serves as a Fortran-friendly interface to the C-bound subprocess structure. More...
 
interface  internal_run
 Interface for synchronously running a subprocess. This generic interface provides a single procedure for executing a command synchronously, waiting for its completion, and retrieving its exit code. More...
 
interface  internal_runasync
 Interface for asynchronously running a subprocess. This generic interface provides a single procedure for executing a command asynchronously, allowing it to run in the background without waiting for completion. More...
